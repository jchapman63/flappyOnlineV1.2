Hi, my name is Jordan and thank you for visiting my website. I'm a third year college student learning how to develop websites in my free time. The aim of this project has been to not only cure my winter break boredom but also help me learn more about user authentication, database inquiries, and animations. The original Flappy Bird game is something I wanted to try out creating in Javascript and I later decided to try developing a larger site out of it. I am happy to have built this and I am excited about the new addition to my personal portfolio.

Game Functionality

  • JavaScript Modules (classes)
  • JavaScript Animation Frames
  • HTML objects with CSS styling

Data Models and Management

  • Python based Django classes and functions
  • Standard built in Django SQL

User Authentication

  • The authentication system is Django’s built in auth system
  • The user profiles are one to one models of Django’s stock user model with useful additions.

Thank you to my roommate and girlfriend for listening to me talk about the website and being the first ones to make accounts and play the game. They both kept me motivated and excited to get the project finished just by being around to listen.
